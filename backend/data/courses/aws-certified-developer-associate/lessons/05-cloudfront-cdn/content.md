# CloudFront CDN

## Overview

Master Amazon CloudFront - AWS's global content delivery network that delivers data, videos, applications, and APIs to users worldwide with low latency and high transfer speeds.

## Alex's Story

PetTracker's user base has gone global! Users in Australia, Europe, and Asia are complaining that pet images load slowly. Alex discovers that serving content from a single US region creates latency issues for international users. Time to learn about CloudFront and edge caching to deliver content globally with blazing-fast performance.

## What You'll Learn

- Create and configure CloudFront distributions
- Understand cache behaviors and invalidation strategies
- Implement signed URLs and signed cookies for secure content
- Configure CloudFront with S3 and custom origins
- Optimize performance with compression and edge caching
- Implement Lambda@Edge for edge computing

## Key Topics

1. **CloudFront Fundamentals** - Distributions, edge locations, and points of presence
2. **Cache Behaviors** - Path patterns, TTLs, and cache policies
3. **Security** - HTTPS, signed URLs, and Origin Access Control
4. **Edge Computing** - Lambda@Edge and CloudFront Functions
5. **Optimization** - Compression, real-time logs, and monitoring

## Prerequisites

- Understanding of S3 (Chapter 4)
- Basic networking concepts
- Familiarity with DNS
