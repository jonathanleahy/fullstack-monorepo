# Homepage Variants V46-V65 Specifications

## Overview
20 new homepage design variants, each targeting specific personas, design aesthetics, and visual styles. All variants follow the core spec requirements:
- Light themes only (no dark backgrounds)
- No purple/violet/indigo colors
- Header with: Logo, Courses, About, Help links, Sign In/Up buttons
- Required sections: Hero, Featured courses/categories, How it works, Testimonials, CTA, Footer

---

## V46: Healthcare Professionals
**Target Audience**: Nurses, doctors, medical staff seeking continuing education
**Design Style**:
- Color palette: Soft teal (teal-500), clean white, light blue accents
- Aesthetic: Clean, clinical, trustworthy
- Shadows: Subtle (shadow-sm), professional feel
- Border radius: Medium (8-12px)

**Tone of Voice**:
- Professional, reassuring
- Evidence-based language
- "Advance your clinical expertise"

**Layout Flow**:
1. Split hero with medical professional image
2. Credential/certification badges bar
3. Course categories (Clinical, Admin, Leadership)
4. Statistics (CE credits earned, professionals trained)
5. Single prominent testimonial from healthcare worker
6. CTA with certification focus
7. Clean footer

**Wildcard Element**: Pulse/heartbeat animation on hover states

## Implementation Notes

All variants should:
1. Import and use `DesignNavigation` component for prev/next navigation
2. Follow the established pattern from existing variants
3. Use Framer Motion for animations
4. Include proper alt text for images
5. Use Unsplash images appropriate to the theme
6. Include the header with Courses, About, Help navigation
7. Be fully responsive
8. Pass TypeScript compilation
